# Source Apps & Assets

| Project/Asset                 | Description                                                                                                                                             |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **assets**                    | Images, OpenAPI specs, and YAML files for agents and APIs.                                                                                              |
| **cs-travel-agent-yaml**      | Declarative agent for travel scenarios using Microsoft 365 Agents Toolkit YAML configuration.                                                           |
| **food-catalog-api**          | RESTful API for food catalog management, built with .NET and includes telemetry integration.                                                            |
| **hr-mcp-server**             | Server for Human Resources agent, integrating with Microsoft 365 Copilot and MCP workflows.                                                             |
| **purchasing-service**        | RESTful API that will be used for managing purchasing processes and supplier interactions demonstrating advanced flows and deep reasoning capabilities. |
| **parts-inventory-connector** | Connector for appliance parts inventory, includes CSV and database assets.                                                                              |
